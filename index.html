SEO Quésako

l'optimisation dans les moteurs de recherches, ou SEO, est un ensemble de techniques qui visent à optimiser la visibilité d'une page web dans les pages de résultats de recherche(les *SERP, pour Search Engie Result pages)

ces techniques cherchent à améliorer la comprehension par les robots d'ineexation de la thématique et du contenu d'une ou de l'ensemble des page d'un site web, et a augmenter le trafic naturel d'un site

L'objectif est d'améliorer le positionnement d'une pas web dans les resultats de recherche sur des mots-clefs ciblés (selon les thèmes principaux du site). on considère que le positionnement d'un site est bien l'orsqu'il est positionné(classé) dans la première page des résultats de recherche, dans l'une des dix premières réponses naturelles d'une recherche sur des mots-clés correspondant précisément a sa thématique

quelque conseil et termes à connaitre :

  1. Le référencement c'est être trouvable avec les mots clés, c'est lui qui detérmine le positionnement. Il est plus que conseillé de veiller à la vérification du trafic généré sur son site.
  2. SEO : search engine optimisation
    • dit gratuit (ou naturel)
    • 4 à 6 mois ayant d'etre référence au naturel
    • il est important d'optimiser le site sur son contenu ainsi que le code qui le rend accessible partout sans oublier les liens et les resaux sociaux
  3. SEA : search engine adversiting
    • Il est payant
    • Achat de mots clefs
  4. le SEM ou search engige marketing regroupe le SEO et le SEA
